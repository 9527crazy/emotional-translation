# UI/UX 设计规范

> v0.1 | 2026-06-22 | Web 端

---

## 1. 设计理念

**温暖 · 沉浸 · 安全 · 简洁**

情绪翻译不是冷冰冰的数据工具，而是帮助人们理解和表达情感的温暖产品。设计应让用户感到被关怀、被理解。

---

## 2. 色彩系统

### 品牌色

| 角色 | 色值 | 用途 |
|------|------|------|
| 主色 | `#6C63FF` | 按钮、强调、品牌标识 |
| 主色-浅 | `#8B85FF` | Hover 状态 |
| 主色-深 | `#4F46E5` | Active 状态 |
| 辅助色 | `#FF6B9D` | 情绪高亮、温暖感 |

### 情绪色彩映射（核心视觉语言）

| 情绪 | 主色 | 浅色(背景) | 深色(强调) | 隐喻 |
|------|------|-----------|-----------|------|
| 😊 快乐 | `#FFD93D` | `#FFF8E1` | `#F9A825` | 阳光 |
| 😢 悲伤 | `#6B9BD2` | `#E3F2FD` | `#1565C0` | 雨水 |
| 😠 愤怒 | `#FF6B6B` | `#FFEBEE` | `#C62828` | 火焰 |
| 😲 惊讶 | `#FF9800` | `#FFF3E0` | `#E65100` | 闪电 |
| 😨 恐惧 | `#9C27B0` | `#F3E5F5` | `#6A1B9A` | 暗夜 |
| 🤢 厌恶 | `#66BB6A` | `#E8F5E9` | `#2E7D32` | 苦涩 |
| 😐 中性 | `#90A4AE` | `#ECEFF1` | `#546E7A` | 平衡 |

混合情绪使用渐变色（如 70% 快乐 + 30% 惊讶 → `linear-gradient(135deg, #FFD93D, #FF9800)`）。

## 6. 核心组件规范

### 情绪仪表盘

```
┌─────────────────────────────────────┐
│         ┌───────┐                    │
│         │  😊   │  主 Emoji 48px     │
│         │  72%  │  脉冲动画          │
│         └───────┘                    │
│                                      │
│  ████████████████░░░░  快乐  72%     │  进度条用对应情绪色
│  ██████░░░░░░░░░░░░░░  惊讶  15%     │  宽度过渡 300ms ease-out
│  ███░░░░░░░░░░░░░░░░░  中性   8%     │
│  ██░░░░░░░░░░░░░░░░░░  悲伤   3%     │
│  █░░░░░░░░░░░░░░░░░░░  愤怒   1%     │
│  █░░░░░░░░░░░░░░░░░░░  恐惧   1%     │
│  █░░░░░░░░░░░░░░░░░░░  厌恶   0%     │
└─────────────────────────────────────┘
```

### 色彩翻译

- 全屏渐变背景，使用当前主导情绪色
- 色彩过渡 CSS transition 500ms
- 可选叠加情绪文字描述

### Emoji 翻译

- Canvas 绘制浮动 Emoji
- 强度越高 → 数量越多、体积越大
- 不同情绪不同动画（快乐=上升，悲伤=下落）

### 情绪曲线图（ECharts）

- 平滑曲线 (smooth: true)
- 最多同时 3 种情绪
- 支持触摸/鼠标交互查看数值
- X 轴时间，Y 轴强度 0-100%

## 7. 动效规范

| 元素 | 动画 | 时长 | 缓动 |
|------|------|------|------|
| 情绪条宽度 | 宽度变化 | 300ms | ease-out |
| 背景色渐变 | 颜色过渡 | 500ms | ease-in-out |
| Emoji 切换 | 缩放+淡入 | 200ms | spring |
| 主 Emoji | 脉冲呼吸 | 2s | infinite |
| 页面进入 | 淡入+上滑 | 300ms | ease-out |
| 面板展开 | 底部滑入 | 250ms | ease-out |

## 8. 交互规范

### 摄像头授权流程

```
[欢迎页] → [点击开始] → [浏览器权限弹窗]
                    ┌────────┴────────┐
               [授权成功]         [授权失败]
                    ↓                  ↓
            [检测界面]         [引导提示 + 手动选择]
```

### 手动情绪选择（降级）

摄像头不可用时提供 Emoji 按钮网格选择情绪。

---

## 9. 无障碍

- 所有交互元素支持键盘 Tab/Enter
- ARIA 标签描述情绪状态变化
- 文字与背景对比度 ≥ 4.5:1
- 颜色不是唯一信息载体（配合 Emoji/文字）
- 尊重 `prefers-reduced-motion`
- 支持浏览器字体缩放不破版
